# Interactive Museum – Virtual Exhibition 🏛️🔍

**Interactive Museum** lets you wander through virtual rooms filled with vintage technological artefacts (pagers, cassettes, early computers). Built during the **Prague Hackathon 2026** by three students from **Weilovka School**.

---

## ✨ Features
- **Immersive navigation** – left/right arrows rotate your view 90°.
- **Dynamic exhibits** – click any object to see its name, description, and image.
- **SPA‑like experience** – smooth transitions without page reloads.
- **Responsive design** – works on desktop, tablet, and mobile.

---

## 🛠️ Built With
- **Backend:** Django, Django REST Framework, PostgreSQL
- **Frontend:** HTML5, CSS3, vanilla JavaScript
- **Deployment:** Render, Gunicorn, WhiteNoise
